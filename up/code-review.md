# Code Review
## 内容
* 代码风格：确保代码风格符合规范
* 代码质量
  * 通用
    * 命名：变量，函数等的
    * 尽可能少的重复代码
    * 是否在必要的地方加入了注释
  * HTML
    * 结构是否合理
    * 标签语意化
  * CSS
    * 是否选择了合适的选择器
    * 是否会污染其他样式
  * JS
    * 一个函数只做一件事情
* 逻辑实现
  * 明显的错误
  * 不合理的设计
